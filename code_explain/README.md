## 使用说明

1. 每个章节都包含两个主要文件：
   - `README.md`：详细的说明文档
   - `*.py`：可运行的示例代码

2. 建议学习顺序：
   - 按章节顺序学习
   - 先阅读README文档理解概念
   - 然后运行和实验代码示例
   - 完成每章末尾的练习题

3. 运行代码示例：
   ```bash
   python chapter1/basics.py
   ```

## 环境要求

- Python 3.8+
- 建议使用虚拟环境
- 主要依赖包：
  - pydantic
  - PyYAML
  - aiohttp
  - psutil
  - prometheus_client
